    Class Client

    Class that runs the drawing application on the client side. It connects to server upon user's request and runs the gui for the application.

    Methods

    NameDescription
    changeBrushColor (color) Helper method that changes color of paint brush
    changeBrushSize (amount) Helper function that changes the size of the brush
    connectToServer () Helper method that asks user for IP address and port number of server they want to connect. If the user incorrectly inputs for more than 5 times, the application ends. If the server is closed, it asks the user if they want to start the application offline.
    createButtons () Method that creates the buttons on the upper margin of the window surface.
    createDrawingStrategy (strokeType) Method that creates instances of paint brush classes. Based on factory design pattern.
    draw (xPos, yPos, r, g, b, brushType, brushSize) Method that draws pixel on x, y coordinate based on given color, paint brush type, and brush size This method uses factory design pattern.
    erase () Method that erases pixels on the gui.
    findPoint (x1, y1, x2, y2, x, y) Helper method that checks if the mouse click is within the boundaries of the button surface
    ifButtonsClicked (xPos, yPos) Method that selects an operation to be performed based on the button that is clicked.
    receiveDataFromServer () Method that is spawned to receive packets from server
    redo () Method that redoes the most recent undone command by repainting the pixels that were erased
    run () Method that runs the application. It runs the application both offline and online. If online, it connects to the server, spawns a thread that listens to the server, and sends packets on a mouse event (draw) If offline, it draws pixels onto the gui. It listens to button click events to manipulate color, brush size, brush types, undo, redo, and erase.
    setUpServer () Method that asks if the user wants to start the application online or offline
    undo () Method that undoes the most recent command by erasing the most recent brush mark
